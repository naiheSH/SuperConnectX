# SuperConnectX 版本历史

## v1.2.1 (2026-07-13)

### 新增功能
1. **系统密钥链加密** - 密码加密方案升级为 Electron safeStorage，Windows/macOS 使用操作系统原生密钥链加密存储，Linux 回退到 AES-256-GCM 加密，安全性大幅提升
2. **日志归档功能** - 新增日志手动归档功能（rotateLogFile），可将当前日志归档保留旧文件，同时创建新日志文件继续写入，方便日志管理和回溯

### 优化修复
1. **Telnet 连接修复** - Telnet 连接不再强制要求输入用户名，简化连接流程；修复 Telnet 点击断开后实际未断开的问题，确保断开操作可靠生效
2. **正则表达式卡死修复** - 修复正则表达式输入 `|` 字符导致应用卡死的问题，增加退化正则检测（以 `|` 开头/结尾、连续 `||`）和空匹配保护，防止死循环
3. **串口显示优化** - 串口侧边栏端口名称固定宽度并省略截断，备注信息独立行显示并支持 tooltip 悬停查看，标签栏备注同样增加省略号和 tooltip 支持
4. **类型统一** - 统一 sessionId 为 string 类型，消除 number/string 混用导致的类型不一致问题
5. **编译修复** - 修复 TypeScript type check 编译报错

## v1.2.0 (2026-07-12)

### 新增功能
1. **分屏显示功能** - 支持拖拽分屏，可将终端区域拆分为多窗口同时查看不同连接内容，提升多任务操作效率
2. **拖拽排序功能** - 支持选项卡拖拽排序和命令列表拖拽排序，方便自由调整连接和命令的显示顺序
3. **深浅模式支持** - 支持浅色/深色主题切换，适配不同使用环境和视觉偏好
4. **命令编辑优化** - 新增命令改为直接在表格内插入编辑，无需弹出对话框，操作更流畅；自动换行功能添加快捷键入口

### 优化修复
1. **UI 全面优化** - 颜色统一规范化，托盘图标变清晰、右键菜单样式优化，高亮下拉列表选中颜色更明显，侧边栏卡片悬浮不再浮动，tooltip 交互逻辑优化
2. **消息提示改进** - el-message 组件国际化适配并精简冗余消息，ElMessage 改为半透明可穿透点击，设置菜单弹出点击外部区域可正常关闭
3. **连接状态修复** - 修复串口关闭选项卡后左侧串口仍显示连接中的状态不同步问题，未连接状态下命令输入保持使能可用
4. **架构重构** - app.vue 和 ipcConnector 模块重构，优化代码结构，提升可维护性
5. **其他修复** - 修复任务栏图标显示问题、编码及测试报错，语法高亮仅在修改时才保存，补全测试用例，更新多语言 README

### 新增功能
1. **数据导出功能** - 支持将设置、命令组、预设命令、串口配置、连接等数据导出为 ZIP 文件，方便数据迁移和备份
2. **数据导入功能** - 支持从 ZIP 文件导入数据，支持增量导入（连接去重）和覆盖导入（设置/串口），导入后自动刷新界面

### 优化修复
1. **用户目录架构重构** - 统一使用 Electron userData 目录存放所有用户数据（日志、备份、终端日志），优化 Chromium 缓存路径避免污染数据目录，首次启动时自动迁移 EXE 同目录的旧数据
2. **FTP 服务端 PASV 修复** - 优化 FTP 服务端 PASV 模式 URL 解析逻辑，用户可手动配置 host 用于 NAT 穿透场景，未配置时由 ftp-srv 自动处理（避免本地连接时数据通道失败）
