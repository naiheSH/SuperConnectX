# SuperConnectX 版本历史

## v1.0.7 (2026-05-17)

### 新增功能
1. **快捷键功能** - 支持自定义快捷键，可切换标签、打开命令编辑器等
2. **设置页面** - 添加设置及持久化功能，支持多种配置选项
3. **系统托盘** - 实现关闭后最小化到托盘功能
4. **日志功能** - 支持日志分片功能，支持日志另存为
5. **串口设置** - 实现串口COM设置，支持波特率可修改可持久化
6. **字体设置** - 支持字体更改及持久化，支持滚动调节字体大小
7. **时间戳** - 添加时间戳配置，支持发送和接收数据显示时间戳
8. **界面布局** - 支持split调节侧边栏，侧边栏展开状态支持记忆
9. **显示切换** - 添加 hex/str 显示切换
10. **串口增强** - 支持串口 crlf、hex功能，支持串口备注功能
11. **命令功能** - 支持命令循环发送，添加编辑命令选项卡
12. **分组管理** - 支持串口分组，支持新建组时拷贝组命令
13. **多标签** - 支持同时打开多个连接，支持多标签操作
14. **其他** - 自定义host支持，新增 docker 脚本，支持 FTP 新增与保存

### 优化修复
1. **稳定性** - 修复多处编译报错、构建报错及编译告警
2. **修复** - 修复hex发送接收异常、编辑备注功能异常
3. **修复** - 修复关闭全部串口、编辑命令同步问题
4. **修复** - 修复新建组问题、显示日志问题、另存逻辑
5. **修复** - 修复打开断联、发送错误等问题
6. **UI优化** - 优化滚动条、按钮样式、输入框、表格等界面显示
7. **优化** - 优化快捷键、固定逻辑、连接逻辑、串口配置
8. **优化** - 无选项卡时显示logo，无连接时选项卡隐藏
9. **优化** - 组件化重构，com口复用通用组件
10. **其他** - 降低安装包大小，修复多处界面显示问题
